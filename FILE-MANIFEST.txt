KEY FINDINGS SUMMARY
====================

✅ Plugin API is production-ready for external server sync
✅ iOS community plugins fully supported (with HTTPS-only constraint)
✅ Two proven patterns: Remotely Save (push) + LiveSync (CouchDB)
✅ Custom REST server viable using event-driven + polling pattern
✅ Tailscale iOS integration works via HTTPS to Tailscale IPs
✅ Development stack: TypeScript + esbuild, fast iteration

⚠️ Critical: Debounce vault.on('modify') events (5-10 sec)
⚠️ Critical: Use requestUrl() not fetch() (CORS)
⚠️ Critical: HTTPS only on iOS (App Store policy)
⚠️ Watch for: Memory leaks (use registerEvent()), requestSave debounce conflicts

Unresolved: iOS background execution, plugin size limits, iOS event batching
